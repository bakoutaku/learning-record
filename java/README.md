# Java

## 概要

HAL東京の授業課題や個人練習で作成したJavaプログラムをまとめています。
Javaの基本文法を復習しながら、入力処理、条件分岐、繰り返し処理、配列、ランダム処理などを学習しています。

## 内容

このフォルダでは、Javaの基礎練習と小さなプログラムを分類して管理しています。

```text
java
├── basic
│   ├── range-sum
│   └── array
└── practice
    ├── student-score-analyzer
    ├── checkout-point-calculator
    └── number-guessing-game
```

## 学習内容

主に以下の内容を学習しています。

* 標準出力
* 変数
* データ型
* 条件分岐
* 繰り返し処理
* 配列
* 入力処理
* 最大値・最小値の判定
* 合計・平均の計算
* break / continue
* Randomクラス
* 基礎的なプログラム作成

## プログラム一覧

| フォルダ                                   | 内容                                |
| -------------------------------------- | --------------------------------- |
| `basic/range-sum`                      | 指定した範囲の整数を合計するプログラム               |
| `basic/array/student-score-statistics` | 配列を使って複数人の点数を集計するプログラム            |
| `practice/student-score-analyzer`      | 学生の点数を入力し、合格人数・最高点・平均点を表示するプログラム  |
| `practice/checkout-point-calculator`   | 商品金額を入力し、合計金額・ポイント・最高金額を計算するプログラム |
| `practice/number-guessing-game`        | ランダムな数字を当てる数字当てゲーム                |

## 学習ポイント

このフォルダでは、Javaの基本文法を使って、コンソール上で動作するプログラムを作成しています。
単純な文法練習だけでなく、入力値のチェック、集計処理、繰り返し実行、途中終了など、実際のプログラムで必要になる処理も意識して練習しています。
