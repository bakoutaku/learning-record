# NumberGuessingGame

## 概要

1から100までのランダムな数字を当てるJavaの数字当てゲームです。
プレイヤーは最大5回まで予想を入力でき、入力した数字が正解より大きいか小さいかを確認しながら答えを探します。

## 使用技術

* Java
* BufferedReader
* Random
* if文
* for文
* while文
* do-while文
* break
* continue
* 入力チェック

## 機能

* 1～100のランダムな数字を生成する
* 最大5回まで数字を予想できる
* 入力された数字が正解より大きい場合、「大きすぎます。」と表示する
* 入力された数字が正解より小さい場合、「小さすぎます。」と表示する
* 正解した場合、「正解です！」と表示してゲームを終了する
* `0` を入力すると、その回をパスできる
* `-1` を入力すると、ゲームを中止できる
* 5回以内に正解できなかった場合、正解を表示する
* ゲーム終了後、もう一度遊ぶか選択できる

## 実行例

```text
１から１００までの数字を試しましょう！
（-1を入力して諦める。）
（0を入力してパスする。）
1回目の予想を入力してください：50
大きすぎます。
2回目の予想を入力してください：25
小さすぎます。
3回目の予想を入力してください：35
正解です！
もう一度遊びますか？ yes / no：no
終了します。
```

## 学習ポイント

このプログラムでは、`Random` クラスを使ってランダムな数字を生成し、`for文` で予想回数を制限しています。
また、`while文` を使って入力値をチェックし、`break` と `continue` を使ってゲームの中止やパス処理を実装しています。

さらに、`do-while文` を使うことで、ゲーム終了後にもう一度遊ぶかを確認し、必要に応じて同じ処理を繰り返せるようにしています。
