# CheckoutPointCalculator

## 概要

商品金額を入力し、購入商品数・合計金額・合計ポイント・最高金額を表示するJavaプログラムです。
100円につき1ポイントとしてポイントを計算します。

## 使用技術

* Java
* BufferedReader
* if文
* for文
* while文
* do-while文
* break
* continue
* 入力チェック

## 機能

* 商品数を入力する
* 商品数は1～20個の範囲で入力チェックを行う
* 各商品の金額を入力する
* 金額は0～50000円の範囲で入力チェックを行う
* `-1` が入力された場合、会計を途中で中止する
* 購入商品数を表示する
* 合計金額を表示する
* 合計ポイントを表示する
* 最高金額を表示する
* 100円未満の商品はポイント対象外として処理する
* 実行後、もう一度会計するか選択できる

## 実行例

```text
商品数を入力してください：3
1個目の金額を入力してください：120
1ポイント追加しました。
2個目の金額を入力してください：80
ポイント対象外です。
3個目の金額を入力してください：250
2ポイント追加しました。
購入商品数：3個
合計金額：450円
合計ポイント：3ポイント
最高金額：250円
もう一度会計しますか？ yes / no：no
終了します。
```

## 学習ポイント

このプログラムでは、`while文` を使って入力値のチェックを行い、`for文` を使って複数の商品金額を処理しています。
また、`break` を使って `-1` が入力された場合に会計を途中終了し、`continue` を使って100円未満の商品をポイント計算から除外しています。

さらに、`do-while文` を使うことで、処理の最後に「もう一度会計するか」を確認し、必要に応じて同じ処理を繰り返せるようにしています。
