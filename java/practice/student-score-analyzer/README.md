# StudentScoreAnalyzer

## 概要

学生の人数と点数を入力し、合格人数・最高点・合格者平均を表示するJavaプログラムです。
点数が60点以上の場合は合格、60点未満の場合は不合格として判定します。

## 使用技術

* Java
* BufferedReader
* if文
* for文
* while文
* do-while文
* break
* continue
* 入力チェック

## 機能

* 学生人数を入力する
* 学生人数は1～30人の範囲で入力チェックを行う
* 各学生の点数を入力する
* 点数は0～100点の範囲で入力チェックを行う
* `-1` が入力された場合、途中で入力を終了する
* 60点以上を合格、60点未満を不合格として判定する
* 入力人数を表示する
* 合格人数を表示する
* 最高点を表示する
* 合格者の平均点を表示する
* 実行後、もう一度処理を行うか選択できる

## 実行例

```text
学生人数を入力してください：3
1人目の点数を入力してください：80
1人目：合格です。
2人目の点数を入力してください：45
2人目：不合格です。
3人目の点数を入力してください：90
3人目：合格です。

入力人数：3人
合格人数：2人
最高点：90点
合格者平均：85点
もう一度実行しますか？ yes / no：no
終了します。
```

## 学習ポイント

このプログラムでは、`while文` を使って入力値のチェックを行い、`for文` を使って複数人分の点数を処理しています。
また、`continue` を使って不合格者の処理を分岐し、`break` を使って `-1` が入力された場合に途中終了できるようにしています。

さらに、`do-while文` を使うことで、プログラムの最後に「もう一度実行するか」を確認し、必要に応じて同じ処理を繰り返せるようにしています。
