# Student Subject Statistics

Javaの二次元配列を使った成績管理プログラムです。

## 内容

3人分・4科目分の点数をキーボードから入力し、以下の内容を計算して表示します。

* 各学生の合計点
* 各学生の平均点
* 各学生の総合合格判定
* 各科目の平均点
* 総合合格者数
* 60点未満の点数の数
* 全体の最高点
* 合計点が一番高い学生

## 使用している主な知識

* 二次元配列 `int[][]`
* 一次元配列 `String[]`
* `for`文
* 二重ループ
* `if`文
* `boolean`型
* キーボード入力 `BufferedReader`
* 合計・平均・最大値の計算

## 判定条件

総合合格の条件は以下の2つです。

1. すべての科目が60点以上
2. 平均点が70点以上

どちらか一方だけでは総合合格にはなりません。

## 入力例

```text
1人目の国語：80
1人目の数学：75
1人目の英語：90
1人目のJava：88
2人目の国語：55
2人目の数学：68
2人目の英語：72
2人目のJava：60
3人目の国語：95
3人目の数学：92
3人目の英語：88
3人目のJava：100
```

2人目は総合不合格です。

3人目の合計点は375です。
3人目の平均点は93.75です。
3人目は総合合格です。

国語の平均点は76.66666666666667です。

* `Hairetu02.java`

## メモ

二次元配列では、外側の添字 `i` を学生番号、内側の添字 `j` を科目番号として扱っています。

```java
scores[i][j]
```

このプログラムでは、`scores[i][j]` は「i人目のj番目の科目の点数」を表します。
