# StudentScoreStatistics

## 概要

6人分の点数を入力し、判定結果・合計点・平均点・最高点・最低点などを表示するJavaプログラムです。
配列を使って複数人の点数を管理し、for文とif文を組み合わせて成績を集計します。

## 使用技術

* Java
* BufferedReader
* 配列
* for文
* if文
* double型
* 最大値・最小値の判定
* カウント処理

## 機能

* 6人分の点数を入力する
* 入力された点数を配列に保存する
* 点数に応じて A / B / C / 不合格 を判定する
* 合計点を表示する
* 平均点を表示する
* 最高点を表示する
* 最低点を表示する
* 合格者数を表示する
* 不合格者数を表示する
* 90点以上の人数を表示する
* 平均点以上の人数を表示する

## 判定基準

|    点数 | 判定  |
| ----: | --- |
| 90点以上 | A   |
| 80点以上 | B   |
| 60点以上 | C   |
| 60点未満 | 不合格 |

## 実行例

```text
6人の点数を入力してください。
1人目：95
2人目：82
3人目：70
4人目：58
5人目：90
6人目：64
1番目の点数は95です。判定：A
2番目の点数は82です。判定：B
3番目の点数は70です。判定：C
4番目の点数は58です。判定：不合格
5番目の点数は90です。判定：A
6番目の点数は64です。判定：C
合計点は459です。
平均点は76.5です。
最高点は95です。
最低点は58です。
合格者数は5人です。
不合格者数は1人です。
90点以上の人数は2人です。
平均点以上の人数は3人です。
```

## 学習ポイント

このプログラムでは、配列を使って複数の点数をまとめて管理しています。
また、for文を使って配列の中身を順番に処理し、合計点・最高点・最低点・人数の集計を行っています。

さらに、平均点を計算したあと、もう一度配列を確認することで、平均点以上の人数を数えています。
配列・繰り返し処理・条件分岐を組み合わせた基礎的な成績集計プログラムです。
