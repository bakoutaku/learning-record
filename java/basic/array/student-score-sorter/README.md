# StudentScoreSorter

## 概要

5人分の名前と点数を入力し、点数が高い順に並べ替えて表示するJavaプログラムです。

`String`型の配列で名前を管理し、`int`型の配列で点数を管理します。  
点数を並べ替える時に名前も同時に入れ替えることで、名前と点数の対応関係を保ったまま表示します。

また、最高点・最低点・平均点も計算して表示します。

## 使用技術

* Java
* BufferedReader
* 配列
* String型
* int型
* for文
* 二重for文
* if文
* 型変換
* ソート処理
* double型

## 機能

* 5人分の名前を入力する
* 5人分の点数を入力する
* 入力された名前を配列に保存する
* 入力された点数を配列に保存する
* 点数が高い順に並べ替える
* 名前と点数の対応関係を保ったまま表示する
* 最高点を表示する
* 最低点を表示する
* 平均点を表示する

```text
