# ProductSalesRanking

## 概要

5個の商品情報を入力し、売上が高い順に並べ替えて表示するJavaプログラムです。

商品名・単価・販売数を入力し、単価と販売数から売上を計算します。
売上が同じ場合は、販売数が多い商品を先に表示します。

また、合計売上・平均売上・最高売上・最低売上も表示します。

## 使用技術

* BufferedReader
* 配列
* String型
* int型

* if文
* 型変換

## 機能

* 5個の商品名を入力する
* 5個の商品の単価を入力する
* 5個の商品の販売数を入力する
* 単価と販売数から売上を計算する
* 売上が高い順に並べ替える
* 売上が同じ場合は、販売数が多い順に並べ替える
* 商品名・単価・販売数・売上をランキング形式で表示する
* 合計売上を表示する
* 平均売上を表示する
* 最高売上を表示する
* 最低売上を表示する

## 並べ替えの条件

| 優先順位 | 条件                  |
| ---: | ------------------- |
|    1 | 売上が高い順              |
|    2 | 売上が同じ場合は、販売数が多い順    |
|    3 | 売上と販売数が同じ場合は、入力順のまま |

## 学習ポイント

このプログラムでは、商品名・単価・販売数・売上を別々の配列で管理しています。

```java
String[] names = new String[5];      // 商品名
int[] prices = new int[5];           // 単価
int[] quantities = new int[5];       // 販売数
int[] sales = new int[5];            // 売上
```

同じ添字の要素は、同じ商品の情報を表しています。

例えば、`names[0]`、`prices[0]`、`quantities[0]`、`sales[0]`は、すべて1つ目の商品に関する情報です。

そのため、並べ替えを行う時は、売上だけでなく、商品名・単価・販売数も同時に入れ替える必要があります。

```java
int tmpSales = sales[i];
sales[i] = sales[j];
sales[j] = tmpSales;

String tmpName = names[i];
names[i] = names[j];
names[j] = tmpName;

int tmpPrice = prices[i];
prices[i] = prices[j];
prices[j] = tmpPrice;

int tmpQuantity = quantities[i];
quantities[i] = quantities[j];
quantities[j] = tmpQuantity;
```

また、並べ替えの条件では、売上を最優先に比較し、売上が同じ場合だけ販売数を比較しています。

```java
if (sales[j] > sales[i] ||
   (sales[j] == sales[i] && quantities[j] > quantities[i])) {
```

この条件により、売上が高い順に並べ替え、売上が同じ場合は販売数が多い順に表示できます。

配列・二重for文・条件分岐・複数配列の同時交換を組み合わせた、基礎的な売上ランキングプログラムです。
