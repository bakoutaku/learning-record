# RangeSumCalculator

## 概要

指定した範囲の整数を合計するJavaプログラムです。
開始の整数と終了の整数を入力すると、その範囲内にある整数の合計を計算して表示します。

## 使用技術

* Java
* BufferedReader
* for文
* if文
* int型
* 入力処理

## 機能

* 開始の整数を入力する
* 終了の整数を入力する
* 開始値から終了値までの整数を合計する
* 開始値が終了値より大きい場合でも、正しく範囲を入れ替えて計算する

## 実行例

```text
指定した範囲の整数を合計します。
開始の整数を入力してください：
3
終了の整数を入力してください：
7
3~7の合計は25です。
```

## 学習ポイント

このプログラムでは、`if文` を使って入力された2つの整数の大小を判定し、`for文` を使って指定された範囲の整数を順番に加算しています。
また、`BufferedReader` と `Integer.parseInt()` を使って、キーボードから入力された文字列を整数に変換する処理を練習しています。
