# HTML / CSS

## 概要

HAL東京の「WF15 フロントエンド制作」の授業で作成したHTML / CSSの学習記録です。
HTMLでWebページの構造を作成し、CSSで見た目やレイアウトを調整する練習を行っています。

## 内容

このフォルダでは、HTML / CSSの基礎練習とページ制作課題を管理しています。

```text
html-css
├── wf15-03-html-css
└── wf15-04-dining-table
```

## 課題一覧

## wf15-03-html-css

HTML / CSSの基本的なタグやレイアウトを学習するための練習データです。

主な学習内容：

* 見出しタグ
* 段落タグ
* 改行タグ
* マークアップ
* div と span
* 画像表示
* テキスト回り込み
* テーブル
* margin と padding
* Flexbox

## wf15-04-dining-table

「ある日の食卓」をテーマにした1ページ構成のWebページです。
画像、文章、リスト、2カラムレイアウトを使い、HTML / CSSによるページ制作を練習しました。

主な学習内容：

* 画像と文章の配置
* main と aside を使った構成
* Flexboxによる2カラムレイアウト
* 背景画像の設定
* 見出しデザイン
* 余白の調整

## 使用技術

* HTML
* CSS

## 学習ポイント

HTMLでページの構造を作成し、CSSでデザインやレイアウトを調整する基礎を学習しました。
特に、外部CSSファイルの読み込み、画像パスの指定、Flexboxによる横並びレイアウトを意識して制作しています。
