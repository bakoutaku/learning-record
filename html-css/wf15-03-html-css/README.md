## 概要

HAL東京の「WF15 フロントエンド制作」の授業で作成した、HTML / CSSの基礎練習データです。

## 学習ポイント

この練習では、`index.html` を目次ページとして作成し、各学習ページへリンクできるようにしました。
各ページでは、HTMLの基本タグやCSSの指定方法を確認しながら、Webページ制作の基礎を学習しています。

特に、以下の内容を意識して作成しました。

* HTMLの基本構造を理解する
* 見出し、段落、改行などの基本タグを使う
* `div` と `span` の違いを理解する
* 画像をページ内に表示する
* テーブルを作成する
* margin と padding の違いを理解する
* Flexboxを使ってレイアウトを調整する
* CSSファイルを外部ファイルとして読み込む
