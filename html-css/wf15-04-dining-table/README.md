# WF15 Dining Table Page

## 概要

HAL東京の「WF15 フロントエンド制作」の授業で作成したHTML / CSS課題です。
「ある日の食卓」をテーマに、食事の写真と説明文を配置した1ページ構成のWebページを制作しました。

## 使用技術

* HTML
* CSS

## 内容

このページでは、食事に関する3つの記事をメインコンテンツとして配置し、右側に補足情報として「よく作った料理」と「好きな料理」のリストを表示しています。

主な内容は以下の通りです。

* メインビジュアルの表示
* 食事ごとの記事作成
* 画像と説明文の配置
* 箇条書きリストの作成
* Flexboxによる2カラムレイアウト
* 背景画像と背景色の設定
* 余白や見出しデザインの調整

## 学習ポイント

この課題では、HTMLでページ全体の構造を作成し、CSSで見た目やレイアウトを調整しました。

特に、`main` と `aside` を使ってページを左右に分け、Flexboxによって2カラムレイアウトを作成した点を意識しました。
また、画像の配置、背景画像の指定、見出しの装飾、余白の調整など、Webページ制作の基本的な要素を確認しました。

## 工夫した点

* `wrapper` を使ってページ全体の幅を固定し、中央に配置しました。
* `main` と `aside` をFlexboxで横並びにしました。
* 記事ごとに画像と説明文をまとめ、読みやすい構成にしました。
* 見出しに左線と下線を付けて、内容の区切りが分かりやすくなるようにしました。
* 背景画像と背景色を設定し、ページ全体の雰囲気を調整しました。
