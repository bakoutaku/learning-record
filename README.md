# learning-record

## 概要

HAL東京での授業課題や個人学習で作成したコードをまとめたリポジトリです。
Java、HTML/CSSを中心に、学習内容を整理しながら記録しています。

## 内容

このリポジトリでは、以下の学習内容を分類して管理しています。

```text
learning-record
├── java
└── html-css
```

## Java

Javaの授業課題や練習プログラムをまとめています。

主な学習内容：

* 標準出力
* 変数
* 条件分岐
* 繰り返し処理
* 配列
* 入力処理
* ランダム処理
* 基礎的なプログラム作成

## HTML / CSS

フロントエンド制作の授業課題や練習データをまとめています。

主な学習内容：

* HTMLの基本構造
* 見出し・段落・改行などの基本タグ
* 画像表示
* テーブル
* margin と padding
* Flexbox
* 2カラムレイアウト
* CSSによる装飾

## 使用技術

* Java
* HTML
* CSS
* GitHub

## 目的

授業で学んだ内容を復習し、基礎的なプログラミング力とWeb制作スキルを身につけることを目的としています。
また、学習記録を整理することで、自分の成長や理解度を確認できるようにしています。

## 今後の予定

今後は、JavaScriptやデータベースの学習内容も追加し、より実践的なWebアプリケーション制作につなげていく予定です。
